= Princess Diaries web app

This app is a simple blogging site made in rails with backbone.js. It was ported to meteor.js - github: https://github.com/JohnSZhang/Crater.git

== To run locally
* download zip-file or run <tt>git clone https://github.com/nnakagaki/facebook-clone.git</tt>
* <tt>bundle install</tt>
* <tt>bundle exec rake db:create db:migrate</tt>
* <tt>rails server</tt>
* go to localhost:3000 to view page
